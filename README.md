## 🖥️ Funcionalidades por Páginas

### **Sobre**
Página inicial com apresentação pessoal, biografia e habilidades técnicas com níveis de conhecimento em Back-end, Front-end e Ferramentas.

### **Formação**
Seção com informações sobre educação, cursos realizados, idiomas com níveis de proficiência e formações acadêmicas atualizadas.

### **Portfólio**
Galeria de projetos realizados com descrição, tecnologias utilizadas, links para repositórios e demonstrações ao vivo.

### **Contato**
Formulário SIMULADO com validação obrigatória em JavaScript:
- ✅ Validação de campos (Nome, Email, Mensagem)
- ✅ Validação de formato de email (regex)
- ✅ Simulação de envio com feedback visual
- ✅ Mensagens de confirmação com Toast
- ✅ Prévia de currículo em PDF com opção de download

## ⚙️ Como Rodar Este Projeto Localmente

### Pré-requisitos
- Git
- Navegador web moderno

### Instalação

```bash
# Clone o repositório
git clone https://github.com/gabicastrum/uninter-portfolio-web.git

# Acesse a pasta do projeto
cd uninter-portfolio-web

# Abra o arquivo index.html em um navegador
# Opção 1: Clique duplo no arquivo
# Opção 2: Use um servidor local (Live Server no VS Code)
```

### Usando Live Server (Recomendado)

Se estiver no Visual Studio Code:

1. Instale a extensão "Live Server"
2. Clique com botão direito em `index.html`
3. Selecione "Open with Live Server"
